# Manuais da secretaria (corpus do RAG)

Textos **Markdown** (ou texto simples) usados pelo **chat de ajuda com RAG**, alinhados ao MVP.

## Pastas por idioma

- **`pt/`** — manuais em **português** (padrão da escola).
- **`en/`** — mesmos temas em **inglês** (acesso internacional / portfolio).

Um arquivo por tema em cada pasta (ex.: `pt/matricula.md` e `en/enrollment.md`) facilita manutenção. Quando mudar a regra no spec, atualiza **os dois** quando o manual cobrir a mesma funcionalidade.

O conteúdo deve refletir os **specs**; se divergir, corrige o manual **ou** o spec.

Ver `docs/specs/ai/chat-ajuda-rag.spec.md` e `docs/language-policy.md`.
